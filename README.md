# 🚀 Dynamic Pathfinding Agent
### A* and Greedy Best-First Search Visualization (Tkinter GUI)

---

## 📌 Project Overview

The **Dynamic Pathfinding Agent** is a Python-based GUI application that visualizes and compares two heuristic search algorithms:

- ✅ A* (A-Star Search)
- ✅ Greedy Best-First Search (GBFS)

The system allows users to create grid environments, place obstacles, enable dynamic obstacle spawning, and observe real-time search behavior with performance metrics.

This project demonstrates how heuristic search algorithms perform in both static and dynamic environments.

---

## 🎯 Features

- Interactive grid environment (customizable size)
- Manual wall drawing and removal
- Random map generation (adjustable density)
- A* Search implementation
- Greedy Best-First Search implementation
- Manhattan & Euclidean heuristics
- Step-by-step animated search visualization
- Dynamic obstacle spawning mode
- Automatic re-planning when path is blocked
- Performance metrics display:
  - Nodes Visited
  - Path Cost
  - Execution Time
  - Re-plan Count
- Keyboard shortcuts for faster interaction
